# Bijvoets Deuren

Statische bedrijfswebsite voor Bijvoets Deuren, gepubliceerd via GitHub Pages op:

https://bijvoetsdeuren.nl/

## Publiceren

Upload alle bestanden en mappen uit deze map naar de hoofdmap van de repository `QuintBijvoets/BijvoetsDeuren`. De meegeleverde GitHub Actions-workflow publiceert de website automatisch.

## Contactgegevens aanpassen

Zoek in `index.html`, `privacy.html` en `script.js` naar `quintbijvoets@hotmail.com` om het contactadres te wijzigen. Voeg alleen bedrijfsgegevens toe die door de eigenaar zijn bevestigd.
